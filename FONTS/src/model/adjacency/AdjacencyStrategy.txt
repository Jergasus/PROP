INTERFACE: AdjacencyStrategy
Package: model.adjacency
Layer: Model

Purpose:
  Strategy pattern interface for cell adjacency. Decouples the Board from
  any specific grid geometry. Four implementations exist: SquareAdjacencyStrategy,
  SquareFullAdjacencyStrategy, HexagonalAdjacencyStrategy, TriangleAdjacencyStrategy.

Methods:
  - getNeighbors(pos, rows, cols): returns the list of positions adjacent to pos
  - areAdjacent(p1, p2, rows, cols): true if p1 and p2 are adjacent.
      Default implementation delegates to getNeighbors(); subclasses may override
      with an O(1) arithmetic check.

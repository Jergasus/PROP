CLASS: Solver
Package: model.algorithms
Layer: Model

Purpose:
  Solves Hidato puzzles using backtracking.
  Uses a clue map (HashMap) to look up already-placed values in O(1) instead of
  scanning the whole board each step. Applies Warnsdorff's heuristic to order
  candidate cells by their number of free neighbours (most constrained first),
  which reduces backtracking significantly.

Public methods:
  - solve(board): fills the board in-place with a valid solution; returns true if found
  - countSolutions(board, limit): counts distinct solutions up to the given limit;
      useful for checking uniqueness (call with limit=2)

CLASS: Validator
Package: model.algorithms
Layer: Model

Purpose:
  Validates Hidato board states, both complete and partial.
  A valid complete solution requires that the numbers 1..N form a path where
  every consecutive pair occupies adjacent cells (per the board's strategy).
  Partial validation checks only that already-placed numbers do not already
  violate the adjacency rule.

Public methods:
  - isValidSolution(board): true if the board is fully filled and forms a valid path
  - isPartiallyValid(board): true if no placed clues contradict the adjacency rules

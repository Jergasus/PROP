Drivers interactius del projecte Hidato.

Contingut:
  SolverDriver.java    - Driver per provar el Solver manualment sobre taulers personalitzats.
  GeneratorDriver.java - Driver per provar el Generator amb paràmetres configurables.
  LevelGenerator.java  - Utilitat per generar i guardar nous fitxers de nivells a data/levels/.

Operativa d'execució:
Totes les accions s'han de realitzar des del directori FONTS/ del lliurament.
1. Compilar: ./compilar.sh (Linux/Mac) o compilar.bat (Windows)
2. Executar un driver:

Linux/Mac:
  java -cp "bin:lib/*" test.drivers.SolverDriver
  java -cp "bin:lib/*" test.drivers.GeneratorDriver
  java -cp "bin:lib/*" test.drivers.LevelGenerator

Windows:
  java -cp "bin;lib\*" test.drivers.SolverDriver
  java -cp "bin;lib\*" test.drivers.GeneratorDriver
  java -cp "bin;lib\*" test.drivers.LevelGenerator

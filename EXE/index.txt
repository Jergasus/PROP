EXE — Executables and Test Suites
===================================
This directory contains all runnable test suites and the interactive driver.
Each subdirectory includes a descripcio.txt explaining its contents.

Subdirectories:
  Solver/        JUnit tests for model.algorithms.Solver + Validator
                 Source: src/test/SolverTest.java (14 tests)

  Validator/     JUnit tests for model.algorithms.Validator
                 Source: src/test/ValidatorTest.java (14 tests)

  DriverHidato/  Interactive driver (SolverValidatorDriver)
                 Predefined Hidato puzzles in hidatos/

To run all JUnit tests at once (from project root):
  javac -cp "lib/*" -d out $(find src -name "*.java")
  java  -cp "out:lib/*" org.junit.runner.JUnitCore test.HidatoTestSuite

To launch the interactive driver:
  ./compilar.sh        (Mac/Linux)
  compilar.bat         (Windows)

EXE — Executable and Unit Test
===================================
This directory contains all runnable unit test and the interactive driver.
Each subdirectory includes a descripcio.txt explaining its contents.

Subdirectories:
  Solver/        JUnit tests for domini.algorithms.Solver (18 tests)

  Validator/     JUnit tests for domini.algorithms.Validator (14 tests)

  DriverHidato/  Interactive driver (SolverValidatorDriver)
                 Predefined Hidatos in hidatos/

To launch the interactive driver go to directory src and execute from terminal:
  ./compilar.sh        (Mac/Linux)
  compilar.bat         (Windows)

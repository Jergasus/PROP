Joc de proves de la classe TriangleAdjacencyStrategy (Test Unitari).

Aquest test comprova la lògica geomètrica de l'estratègia d'adjacència triangular.
S'hi verifiquen:
- El càlcul de veïns per a triangles amb orientació "cap amunt" (suma fila+columna parella), els quals tenen el veí vertical a sota.
- El càlcul de veïns per a triangles amb orientació "cap a baix" (suma fila+columna senar), els quals tenen el veí vertical a dalt.
- Casos extrems en els límits del tauler on certs veïns (com l'esquerre o el superior) queden fora de límits.

No es requereixen fitxers de dades externs.

OPERATIVA D'EXECUCIÓ:
Situeu-vos al directori FONTS/ del lliurament.
1. Executeu el script compilar.sh (Linux/Mac) o compilar.bat (Windows).
2. Executeu el test:

Per a Linux/Mac:
java -cp "bin:lib/junit-4.13.2.jar:lib/hamcrest-core-1.3.jar" org.junit.runner.JUnitCore TriangleAdjacencyStrategyTest

Per a Windows:
java -cp "bin;lib\junit-4.13.2.jar;lib\hamcrest-core-1.3.jar" org.junit.runner.JUnitCore TriangleAdjacencyStrategyTest

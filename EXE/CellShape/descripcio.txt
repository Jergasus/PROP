Joc de proves de l'enumeracio CellShape (Test Unitari).

Aquest test verifica l'enumeracio de les formes permeses per a les celles de l'hidato.
S'hi comprova:
- Que l'enum conte exactament les tres constants definides a la teoria del joc (SQUARE, HEXAGON, TRIANGLE).
- Que els metodes de conversio implicita de String a Enum (valueOf) funcionen correctament, ja que aixo sera critic per a la futura lectura i instanciacio d'hidatos des de fitxers de text pla.

No es requereixen fitxers de dades externs.

OPERATIVA D'EXECUCIO:
Situeu-vos al directori FONTS/ del lliurament.
1. Executeu el script compilar.sh (Linux/Mac) o compilar.bat (Windows) per generar els binaris.
2. Sense canviar de directori, executeu el test amb la comanda corresponent al vostre sistema operatiu:

Per a Linux/Mac:
java -cp "bin:lib/junit-4.13.2.jar:lib/hamcrest-core-1.3.jar" org.junit.runner.JUnitCore CellShapeTest

Per a Windows:
java -cp "bin;lib\junit-4.13.2.jar;lib\hamcrest-core-1.3.jar" org.junit.runner.JUnitCore CellShapeTest
